Global Animation Strategy

  • Scroll trigger: IntersectionObserver (threshold 0.15) — no libraries needed
  • Easing: cubic-bezier(0.16, 1, 0.3, 1) — organic, decelerating feel
  • Duration: 600-900ms for reveals, 200-300ms for micro-interactions
  • Stagger: 60-100ms between sibling elements
  • Theme-aware: All colors reference CSS custom properties — automatic sativa/indica adaptation
  • Mobile: Same animations, slightly shorter durations, no mouse-dependent effects
  • Reduced motion: Instant visibility, no transforms — content appears without animation
  • Zero dependencies: Pure CSS + vanilla JS IntersectionObserver. No framer-motion, no GSAP.
